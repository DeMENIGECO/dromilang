📘 Funzioni di Dromi

Funzioni base per creare interfacce grafiche

🟢 h1(text)

Utilità: Mostra un titolo grande.

h1("Titolo principale")

🟢 h2(text)

Utilità: Mostra un sottotitolo.

h2("Sottotitolo")

🟢 h3(text)

Utilità: Titolo più piccolo.

h3("Titolo piccolo")

🟢 h4(text)

Utilità: Titolo ancora più piccolo.

h4("Titolo mini")

🟢 p(text)

Utilità: Mostra un paragrafo di testo.

p("Questo è un testo")

🟢 button(text)

Utilità: Crea un bottone cliccabile.

button("Cliccami")

🟢 inputer()

Utilità: Crea un campo di input.

inputer()

🟢 clear()

Utilità: Pulisce la schermata.

clear()

🟢 update()

Utilità: Aggiorna la finestra.

update()

🟢 set_window_size(width, height)

Utilità: Imposta la dimensione della finestra.

set_window_size(800, 600)

🟢 set_window_title(title)

Utilità: Cambia il titolo della finestra.

set_window_title("Dromi App")

🟢 run() / dr.init()

Utilità: Avvia l'applicazione Dromi.

run()

🚀 Esempio Completo

set_window_title("Demo")
set_window_size(800, 500)

h1("Benvenuto!")
p("Questa è una app Dromi")

button("Clicca qui")
inputer()

run()
